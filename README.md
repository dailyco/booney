<br/><br/><br/>


## 📜 실행 가이드
### 프로젝트 가져오기
```bash
$ git clone https://github.com/dailyco/booney.git
```
<br/>

### 코드 실행하기
```bash
$ yarn install
$ yarn start
```
코드 실행후 http://localhost:3000 에서 확인 가능합니다!
